Where you are compared with where you want to be, the gap between the two. It's a compelx process, extentive research and a lot to consider. Need to consider every IT aspect, takes a LOT of time specially to gather the data, emails, technical research. 
Choosing the framework : choose the baseline, internal set of goals. Formal standards : NIST, ISO/IEC (information security management systemns), can create your own baseline.
Those baseliens anylyse the people and processes your organisation use. The baseline for employees : 
- formal experience
- current training 
- knowledge of security policies and procedures
Examine the current porcess : research existing IT system, ecaluting existing security policies. 
The comparaison : evaluate the existing system. Identify weaknesses along with the most effective processes. A detailed analysis examine broad security categories, break those into smaller segments.
