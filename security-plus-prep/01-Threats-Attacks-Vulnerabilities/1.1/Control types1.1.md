Security risks are out there : many different categories, we don't only protect data but people and physical objects too
prevent security events, minimize the impact and limite the damage 
# Control categories 
## technical control 
Controls implemented using systems (firewalls, anti-virus)
## Managerial controls 
- Administrive controls associated with security and implementation
- Security policies to explain to people how to manage, standard operating procedures. 
- Managerial controls are therefore implemented in the policies 
# Operational controls 
- Controls implemented by people instead of systems
- Security guards, awarness programs, to explain best practices to IT tam 
## Physical control 
- Limit acces to a room, computer, building (guard shack, fences lock, badge readers)
 
# Control types
- Preventive control : block access to a resource
	- prevent acces by firewall rules(technical), following security policy 8managerial), guard shack checks all identification (operational) or enable door locks (physical)
- Deterrent : discourage an intrusion attempt, make them think twice. Application splash screen threat of demotion, front reception desk or posted waning signs (physical)
- Detective : soesn't prevent acces but gives a warning log, identify an intrusion attempt. 
	- Collect the review system logs, review login reports, property patrols and motion detectors.
- Corrective : apply control after an even occured, reverse the impact of said event and continue operatinf with minimal downtime. 
	- Exemple : restoring from backups to mitigate a ransomware infection, policies to report the security issues, contact law enforcement to manage criminal activity (someone jump the fence), use a fire extinguisher if something catches fire.
- Compensating : control using other means because existing controls are no sufficient, can be temporary. Prevent the exploitation of a weakness. 
	- Firewall blocks a specific application instead of patching the app, 
	- implement a seperation of duties, 
	- requite simultaneous guard duties
	- or generator used after power outage.
- Directive : direct a subject (person) towards sexurity compliance. relatively weak security control
	- Store all sensitive files in a protected folder
	- create compliance policies and procedures
	- train users on proper security policy
	- post a sign for "Authorized Personal Only"
